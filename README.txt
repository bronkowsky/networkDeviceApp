# Aplikacja do zarzadzania urzadzeniami sieciowymi

Ta aplikacja umozliwia:

- Dodawanie i usuwanie urzadzen
- Edycje i zapis
- Zapisywanie danych do pliku JSON
- Filtrowanie i sortowanie danych
- Obsluge wyjatkow i walidacje danych (mozliwe wpisanie tylko IP do liczyb 255.255.255.255 i sprawdzanie duplikacji adres√≥w IP)
